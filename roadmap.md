# RoboSimLab — Development Roadmap

## ✅ Completed — All 11 Modules + Builder

- [x] **Design System** — Dark research-lab theme, JetBrains Mono, semantic tokens
- [x] **Simulation Gallery** — Central hub with 11 module cards
- [x] **PID Control Lab** — Tunable P/I/D gains, response curves, CSV export
- [x] **Robotic Arm Kinematics** — 3D Three.js arm, FK, coordinate frames, transformation matrices
- [x] **Swarm Robotics** — 3D boids flocking with orbit camera, up to 300 agents, depth-sorted rendering
- [x] **Motion Planning Studio** — A* + RRT algorithms, drawable obstacles, animated exploration
- [x] **Sensor & Perception Lab** — Lidar raycasting, depth profile, camera FOV, SLAM mapping
- [x] **Humanoid Balance** — 3D humanoid robot, inverted pendulum PD control, CoM, support polygon
- [x] **RL Playground** — Q-learning grid world, policy arrows, Q-value heatmap, learning curve
- [x] **Robot Dynamics & Torque** — 3D arm with torque/velocity/energy charts, payload & friction controls
- [x] **Autonomous Navigation** — Waypoint following, reactive obstacle avoidance, SLAM mapping
- [x] **Multi-Agent Coordination** — Nearest-first & priority auction task allocation, comm radius
- [x] **Custom Robot Builder** — 1-7 DOF, configurable link length/radius/type, JSON export
- [x] **Data Export** — CSV export for PID simulation data, JSON export for robot configs
